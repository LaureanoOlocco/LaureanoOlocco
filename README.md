# [Laureano Olocco](https://laureanoolocco.github.io)

## Reverse Engineer | Detection Engineer | Systems Programmer

Final-year Computer Engineering student at Universidad Nacional de Cordoba, concurrently pursuing Computer Science for a stronger theoretical foundation.

I'm drawn to the lowest levels of the stack — understanding how systems **break**, how malicious code **behaves**, and how to **detect** it. My work sits at the intersection of static analysis, binary reverse engineering, and detection engineering, building tooling around threats targeting Linux, firmware, and network devices.

---

### Current Focus
- Reverse engineering ELF/PE binaries with Ghidra and static analysis tooling
- Behavioral and structural analysis of malware targeting Linux and network devices
- Writing detection rules and binary-level patterns for threat identification
- UEFI/firmware internals, low-level exploitation, and emulation labs
